# Deep Generative Models -- Course Assignments (Fall 2023)

**School of Electrical and Computer Engineering, College of Engineering,
University of Tehran, Tehran, Iran**

This repository contains my assignment submissions for the **Deep
Generative Models** course offered at the University of Tehran in Fall
2023, instructed by **Dr. Tavassolipour** and **Dr. Sadeghi**.

The assignments cover a broad set of foundational and advanced concepts
in generative modeling, including **probabilistic graphical models,
variational inference, VAEs, normalizing flows, GANs, diffusion models,
and LLM fine-tuning**.
